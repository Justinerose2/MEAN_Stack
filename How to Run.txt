<Notice> MongoDB used in this Folder is connected on Online Server MongoDB(No need to change any Code)

Step 1: Open terminal in VS connected
Step 2: Input >>        npx nodemon index.js

Step 4: Inpute >>       ng serve

(Step 4 will run the angular Front End)
